Conclusiones ¶

Un proyecto con esta complejidad conlleva un tiempo de trabajo mínimo para completar los requisitos además de una cantidad extra para que quede todo bien. Tener que compaginar este tiempo con las prácticas y tener un límite de tiempo tan ajustado hace que sea difícil llevarlo adelante. Aun así, si te gusta y tienes algo de voluntad se hace. También He tenido la oportunidad de aprender cosas, por ejemplo un poco de nodejs, detalles del framework o gestionar el tiempo mejor, y de comprobar otras que ya había aprendido, como que no hay que reinventar la rueda.
